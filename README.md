# npm-users
npm user manager

-----------

## Installation

```javascript
npm i npm-users -g
```

-----------

## Use

```bash
Usage: npm-users [options]


Options:

  a, add <alias>     Add user
  r, remove <alias>  Remove user
  u, use <alias>     Use user
  l, list            List users
  w, whoami          Current logged in user
  -h, --help         output usage information
```
